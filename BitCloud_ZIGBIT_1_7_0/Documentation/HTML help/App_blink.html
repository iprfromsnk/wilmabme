Blink Application

This example introduces the simplest application that uses timer and LEDs. When started, the application makes all the MeshBean's LEDs blink synchronously with certain period.

The frequency of blinking may be varied manually within range specified at compile time. Namely, it increases or decreases by a factor of two on pressing the SW1 or SW2 button on the MeshBean board, respectively. Minimum, maximum and initial period values (in milliseconds) are defined in configuration file.

It is not recommended to set the initial period outside the minimum..maximum range, or set the minimum value to less than 2. In these cases, the application may behave inconsistently. (The corresponding error checking was omitted intentionally in order to keep the application code as simple as possible.)

Configuration parameters

APP_BLINK_PERIODInitial blinking period (ms)
APP_MIN_BLINK_PERIODMinimum blinking period (ms)
APP_MAX_BLINK_PERIODMaximum blinking period (ms)

Description of parameters common for all applications can be found in Sample Applications overview.

The source code and Makefile of Blink application are located at [SDK]\Sample Applications\Blink directory.
