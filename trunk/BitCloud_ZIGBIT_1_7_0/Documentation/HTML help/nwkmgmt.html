Network Management

Network management provides the following capabilities:

  • To retrieve management information from the devices including:
    • Network status
    • Network parameters
    • Node address
    • Energy detection scan results
    • Link quality to neighbor nodes
    • Neighbor table contents
    • Neighbor nodes amount

  • To set management controls including:
    • Start network primitive is generated by coordinator in order to form a network. Routers and end devices shall generate this primitive to join to existing network.
      On receipt of this primitive, device shall return a confirmation.
    • Reset network
    • Leave network.
    • Permit joining
    • Network update and fault notification
    • Set transmitter power
